# 設計書03: 時間構造・練習・イベント

試合はゲーム時間の1割以下。残り9割の「日常」を、作業にせず采配にするための設計。
基本思想: **委任で操作コストを削り、例外と選択で意思決定を残す**。

---

## 1. 時間構造: 週ターン制

- 1ターン = 1週間（年間約50ターン）
- 各ターンの流れ:
```
① イベント処理（あれば） → ② 練習計画の確認/変更 → ③ 週末アクション（練習試合/休養/大会） → ④ 週次リザルト
```
- 練習計画は**テンプレ保存＋「変更するまで継続」**方式。何もなければ1ターン数秒で送れる
- イベント発生時のみ自動停止する「まとめて進行」ボタンを標準装備（元祖の自動操作思想を継承・強化）

## 3. 練習システム

### 3.1 練習メニュー → 能力マッピング
| メニュー | 主効果 | 副効果 |
|---|---|---|
| ミート打撃 | ミート | 選球眼 |
| 長打打撃 | パワー | 弾道 |
| 選球練習 | 選球眼 | ミート |
| 筋力トレ | パワー | 肩（筋肉バカ育成向け） |
| 走塁練習 | 走力 | 盗塁系スキル |
| バント練習 | バント | ミート（バットコントロール） |
| 守備練習 | 守備（地力） | 捕球 |
| ポジション別守備（投/捕/一/二/三/遊/左/中/右） | そのポジションの**守備適性** | — |
| 内野汎用守備 | 内野4ポジの適性を薄く全上げ（ユーティリティ育成） | — |
| 外野汎用守備 | 外野3ポジの適性を薄く全上げ（ユーティリティ育成） | — |
| 遠投・送球 | 肩 | 守備 |
| 投げ込み | コントロール | スタミナ |
| 球速強化 | 球速 | スタミナ・パワー（各少し。コントロールは上げない） |
| 変化球練習 | 球種ランク | 新球種習得判定 |
| ランニング | スタミナ | 走力 |
| ビデオ研究 | 配球・対戦準備 | 精神力（要ビデオカメラ、元祖VTR踏襲） |
| 休養 | 効果なし（時間を割り当てない枠） | — |

**守備の二層**: 守備は「守備地力(Fielding/Catching)」と「9ポジション個別の適性」の二層。試合では *実効守備力 = Fielding × 適性補正(出場ポジション)*（基準適性50で×1.0、本職＝高適性でボーナス、慣れないポジでペナルティ。補正式は `data/coefficients.yaml` fielding: 駆動）。「守備練習」は地力を、「ポジション別／汎用守備」は適性を伸ばす。広く適性を持つ選手＝ユーティリティ。

### 3.2 強度設定 ※廃止（2026-07-17）
- 練習強度（ゆるめ/普通/猛練習）と選手の蓄積疲労は廃止。練習はみんな同じ強度。
  「高校生なんだから気合いで出場」の方針で、疲労という隠しコストを持たない設計に変更した。
- 育成式から疲労係数・強度倍率を除去。休養メニューは効果なし（時間を割り当てない枠）。
- 投手スタミナ（試合内の球数消耗＝PitchingFatigue）は別概念として存続。

### 3.3 個別指導（毎週の中心的な意思決定）
- 監督は毎週**3枠**だけ選手を個別指導できる
- 効果1: 対象能力の経験値×3
- 効果2: **見極め** — その選手の隠しパラメータ（才能上限・成長タイプ・性格）の推定精度が上がる
- 効果3: 信頼関係の構築（性格・精神力の成長に波及）
- 「限られた目を誰に注ぐか」がこのゲームの日常パートの采配になる

### 3.4 委任
- コーチ/部長にメニュー編成を委任可能（方針だけ指定: 打撃強化・守備強化・バランス等）
- 委任の質は監督メタ（設計書04予定）のスタッフ能力に依存

### 3.5 怪我システム
軽微〜重度の**段階制**（選手生命に関わる大故障のような重すぎる離脱は設けない）。
症状は**常に可視**（部位・程度が監督に見える）。葛藤は「見抜けるか」ではなく「この状態で使うか」に集中させる。

- **発生**: 週次の基礎発生率で判定（怪我耐性が隠し補正）。※疲労・猛練習トリガーは廃止（§3.2）に伴い基礎率のみに縮退
- **段階**（例）: 軽度 / 中度 / 重度。段階に応じて能力ダウン幅と回復に要する期間が増す
- **練習**: 怪我中は原則練習不可（患部を使う練習は不可、リハビリ/休養に限定）。放置より休ませる方が早く治る
- **試合**: 出場は**可能**。ただし段階に応じた能力ダウン付き（痛みを押しての出場）
- **「押して出場」の設計**（ジレンマの核）:
  - 怪我を押して使うと**悪化リスク**（段階が上がり離脱が延びる）。基本的には損な選択
  - それでも使いたくなる状況を用意して葛藤にする: どうしても勝ちたい一戦、代えのいないエース/主軸、
    引退のかかった3年夏。勝つために今を取るか、選手の回復を待つか
  - 押して勝った場合の代償（次戦以降のさらなる能力ダウン・離脱延長）を明示し、安易な連発を抑制
- **回復**: 休養・時間経過で段階が下がる。医療施設（設計書04の備品）で回復速度に補正
- **接続**: 能力ダウンは物理層に一律係数で反映（設計書01のパイプラインは不変）。
  段階はUIの選手状態に常時表示（設計書06 §3.1のフィード＋選手詳細）

## 5. イベントシステム

単調さを破る装置。3種類に分類する。

### 5.1 固定イベント（カレンダー連動）
入学式・仮入部→正式入部、中間/期末テスト週、夏合宿・冬合宿、文化祭、
ドラフト会議、プロ入りしたOBの挨拶訪問（元祖踏襲。名声/寄付ボーナス）、卒業式

### 5.2 ランダムイベント
怪我・故障、雨天（グラウンド不可）、マスコミ取材（名声↑・次試合プレッシャー↑）、
プロスカウト視察（対象選手の精神力試練）、OB会からの寄付、部員の体調不良

### 5.3 選択肢イベント（監督の判断で分岐）
- 例:「エースが肘の違和感を隠している」→ 検査に行かせる（数週離脱・信頼↑）/ 様子を見る（故障リスク）
- 例:「主力が練習をサボっている」→ 叱る / 話を聞く / 見逃す（性格との相性で結果分岐）
- 例:「無名の1年が自主練を続けている」→ 個別指導枠を使う / 使わない（晩成型発見の入口）
- 効果は小さな補正＋信頼・性格の変化に留め、根幹は数値管理という元祖の硬派さを守る

### 5.4 学業要素（軽め）
- テスト週（年4回）は練習効率が自動低下
- 赤点者は次の練習試合出場停止。「文武両道」方針でリスク緩和
- ※煩雑になるなら発生を通知のみにするオプションを用意

### 5.5 選手成長イベント（能力の非線形な跳ね）
経験値の積み上げ（設計書02 §5）に加え、能力がぐっと動くイベントで育成に浮き沈みを与える。
上下バランス良く（恒久的な伸び悩みもあり）。ただし**下降は理不尽にしない**原則を厳守。

- **上昇イベント（福音・レア）**:
  - **覚醒**: ある分野の能力が一段跳ね上がる。伸びしろの大きい選手・実戦経験の蓄積・
    好調持続などが条件になりやすい（完全ランダムではなく「兆し」の上に起きる）
  - **フォーム開眼 / 球種のコツを掴む**: 特定能力や新球種がぐっと伸びる
  - **大舞台での化け**: 甲子園など大舞台での好パフォーマンスが精神力・実戦系の飛躍に繋がる
- **下降イベント（試練）**:
  - **スランプ（一時的）**: 一定期間の能力ダウン。立て直せる（起用法・休養・個別指導で回復）
  - **伸び悩み（恒久的）**: 才能上限/伸びしろに達し、その分野がほぼ頭打ちになる。
    ※「理由のない突然の劣化」ではなく「ここが限界だった」という納得のある頭打ちとして提示
  - **イップス系（限定的・恒久寄り）**: 送球や制球など特定動作の不調。克服イベントで取り返す道を残す
  - 故障由来の低下は怪我システム（§3.5）で扱い、ここでは重複させない
- **理不尽回避の原則**:
  - 恒久ダウンは「限界／不調の顕在化」に限定し、丹精込めた選手が脈絡なく劣化する事態は作らない
  - 下降には必ず文脈（限界・不調・怪我）を持たせ、多くは立て直し/克服の余地を残す
  - 上昇はレアな福音、下降は乗り越えられる試練、として非対称に配置
- 実装: イベントはYAML（設計書04 §3のテーブル）で管理。発生条件に選手の隠しパラメータ・調子・
  実戦経験を絡める。検証（設計書07）で「覚醒率」「恒久ダウン率」がプレイ体験を壊さない範囲か確認

### 5.6 気づきイベント（育成眼による隠しパラメータの部分開示）
選手の隠しパラメータ・成長の兆し（§5.5）を、監督の**育成眼**（設計書04）を通じて
"気づき"として通知する。全部は分からない・たまに気づく、が基本。監督が育つほど見える世界が広がる。
- 例:「この選手、最近伸びてきているようだ」（好調＋成長中のサイン）
     「彼に投手の伸びしろはないかもしれない」（投手伸びしろが低いことの示唆）
     「打撃はまだ伸びる余地がありそうだ」（打撃伸びしろが高いことの示唆）
- **育成眼が高いほど**: 気づきの頻度が上がり、精度も上がる（早期に・正確に気づく）
- **育成眼が低いと**: 見逃しが多く、たまに読み違える（断定ではなく「〜かもしれない」の温度で提示）
- 個別指導（設計書03 §3.3の「見極め」）と相補的: 指導は能動的に精度を上げる手段、
  気づきイベントは受動的に降ってくるヒント。両方で選手像が像を結んでいく
- 断定せず含みを持たせる表現にし、外れることもある（＝完全な答え合わせにはしない）ことで
  選手を観察する楽しみを損なわない
