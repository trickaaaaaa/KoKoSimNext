## 0. 背景と決定

- 問題: 現UIは単一色相の暗緑で「のぺっと」見える（面の明度差のみ・枠影なし・色の塊なし）
- 決定: ネオブルータリズム基調の**昼の球場**スキンへ全面転換する
  - 白球の白地＋黒インクの太枠とベタ影＋芝/土の色ブロック＋応援団赤CTA＋空青の操作系＋桜の選択色
  - **スコアボード（トップバー・掲示板部品）は昼でも黒いまま**＝夜スキンとの連続性の錨
  - 選手行など一覧の行は罫線区切りをやめ、**井の上に独立パネルを積む**
  - 人名・校名の太明朝、Oswald数字、DotGothic掲示板は維持（「日本の野球中継」の芯）
- 機能色は不変: ランクS〜G 7色・調子5色は色覚対応の状態コードなので**変更しない**（明地での視認はG0で実測）

## 2. 影と枠のUSS再現（前提: USSに box-shadow / 擬似要素は無い）

| 方式 | 作り | 用途 |

| 案ロ: 背面プレート | wrapper内に absolute の黒プレートを敷き、前面パネルをオフセット | 見た目忠実。カード級・CTA向け。要素数は倍 |

  - `PlateHeader`（ペンキ塗り見出しプレート。grass / dirt / sky の3変種＋白抜き題字）
  - 既存アトム改修: RankChip（2pxインク枠）/ NumberBadge / StatBar / Chip / FilterChip / SegmentedControl / card系 / abil-row / TopBar / nav

## 5. 正典改定案（G4で CLAUDE.md「UI原則」と UI-BUILD-METHOD.md へ反映）

- **②強調は希少資源**: 「アクセント黄は3箇所まで」→「**CTA赤 #E8452E は1画面1箇所**（主要アクション専用）。空青=操作系、桜=選択状態、黄=掲示板内の電光数字専用。警告も赤系のため、警告表示は形（枠・アイコン）で区別する」
- **④区切りは余白と背景色で**: 全面改定→「**区切りはインク枠（2px/3px）とベタ影で作る**。ヘアラインの装飾罫は引き続き禁止。面の明度差による区切りは井（--dirt-well 等）に限る」
- **①密度・③数字・⑤トークン・⑥状態・⑦操作** は現行のまま（数字装飾禁止にベタ影は含めない＝影はパネルの属性であって文字装飾ではない、と明記）
- design-16: 配色・面の節に「design-19 が置換」の注記。書体3役・掲示板部品・ヒラギノ禁止等は存続
- 試合盤面ビューの例外条項（色は状態コード）は維持

## 6. リスク・未決（実装で勝手に決めない）

| 項目 | 論点 |
|---|---|
| 明地の眩しさ | 長時間プレイで白地がきついか。夜スキン併設（トグル）は大工事のため初期は昼のみを推奨 → **要ユーザー判断** |
| ランクF/G・調子「普通」の灰系 | 明地での視認性をG0で実測。必要なら明度のみ再調整（色相は不変） |
| 掲示板部品の浮き | 黒い掲示板が明地でどの程度目立つか。枠の要否をG0で確認 |
| design-18（3D対決ビュー） | ピクセル化シェーダ画面と昼UIの調和は別途議論 |
| フォント+4px未コミット分 | どの時点でコミットするか（G1で本ブランチに取り込む案を推奨） |
